# Hatsune Miku Audio Player

Projeto desenvolvido em Next.js com o objetivo de manipular áudio no navegador.

## Funcionalidades

- Executar áudio
- Pausar áudio
- Controlar o volume
- Exibir status do áudio

## Tecnologias utilizadas

- Next.js
- React
- TypeScript
- CSS
